Lister

I WordMat kan man arbejde med lister

Definer en liste eller Makelist

I WordMat kan man bl.a. arbejde med lister

Vi kan selv definere en liste

$$ { \text{eksList := [ 8 ; 9 ; 5 ; 3 ; 2 ; 3 ] } } $$

Men vi kan også skrive et udtryk som generere en liste

Udtrykket vil være formuleret således:

$$ { \text{listeNavn := makelist(ligning; variabel navn; antal) } } $$

Dette kunne ekempelt være:

$$ { \text{listeNavn := makelist(4x+3; x; 10) } } $$

Hvis vi efterfølgende beregner listen, vil vi få indholdet af listen:

$$ { \text{listeNavn=[7;11;15;19;23;27;31;35;39;43] } } $$

Efterfølgende kan man hente et element i listen. Dette gør man ved hjælp af indeksering. Maxima (og WordMat) bruger 1-baseret indeksering, hvilket betyder, at den første position er 1.

$$ { \text{eksList[2]} } $$

Dette vil retunerer værdien 9.

Andre eksempler som kan afprøves

$$ { \text{makelist(random(100),10)=[4;91;29;85;98;3;35;65;40;26] } } $$
$$ { \text{makelist(concat(x;i),i,6)=[x1;x2;x3;x4;x5;x6] } } $$

Sorter en liste

$$ { \text{sort([ 8 ; 9 ; 5 ; 3 ; 2 ; 3 ])} } $$

Eller, vi kan sorter en liste vi allerede har defineret

$$ { \text{sort(eksList)} } $$

Dette vil i begge tilfælde returnerer en sorteret liste

$$ { \text{sort(eksList)} = [2;3;3;5;8;9]} $$

Længden på en liste

Vi kan også finde antal enheder i listen

$$ { \text{length(eksList)} } $$

Dette returnerer 6, da der er 6 tal i listen

$$ { \text{length(eksList) = 6 } } $$

Sum af lister

Find summen af all værdierne i en liste

$$ { \text{sum(liste[i] ; i ; startIndex ; slutIndex )} } $$

Eksempel

$$ { \text{sum(eksList[i] ; i ; 1 ; 6 )} } $$

Vi kan også skrive det således

$$ { \text{sum(eksList[i] ; i ; 1 ; length(eksList) )} } $$
$$ { \text{sum(andel[i] ; i ; 1 ; 6 ) = 30} } $$

Andre muligheder med lister

Delete indhold fra en liste

Når vi deleter et indhold fra en liste, er vi nød til at deffinere en ny liste med resultatet. Dette er vi nød til pga. måden Maxima (WordMat) refererer til en liste.

$$ { \text{eksList := [ 8 ; 9 ; 5 ; 3 ; 2 ; 3 ] } } $$
$$ { \text{newListTwo := delete(3;eksList) } } $$

Dette fjerner alle eksempler på 3 i listen

$$ { \text{newListTwo=[8;9;5;2] } } $$

Modulus

Modulus operation bruges ofte i programmering og matematik til at finde rester, når man arbejder med cykliske eller gentagne processer (som klokkeslæt eller rotationer).

For at bruge modulus (eller modulo) operationen i WordMat (som bruger Maxima som beregningsmotor) kan du benytte funktionen mod(). Modulus-operationen returnerer resten, når et tal divideres med et andet.

$$ { \text{mod(x ; y)} } $$

Ekempler på brug af modulus

Her returnerer mod(x, y) resten af divisionen af x med y

$$ { \text{mod(495 ; 60)} } $$

Dette vil returnerer en rest på 15

$$ { \text{mod(495 ; 60) = 15} } $$

Funktioner

I Maxima (WordMat) kan man også arbejde med sine egne funktioner

$$ { \text{funktionNavn(variabeler):= regneudtryk} } $$

Ekesmpler på en funktion der udregner hældningstal

$$ { hældningstal( x1 ; y1 ; x2 ; y2):= {(y2 - y1) \over (x2 - x1)} } $$

Her kan vi indsætte to koordinater, hvorefter den udregner hlkdningstallet

$$ { hældningstal( 4 ; 10 ; 6 ; 16) = 2 } $$

Floor, Ceiling og Round

Floor funktionen

floor(x) bruges til at afrunde nedad til det nærmeste heltal.

Dette returnerer det største heltal, der er mindre end eller lig med x.

$$ { \text{floor(48,9)} } $$

Dette vil runde ned til 48

Løsningseksempel på opgave 1 i FSA maj 2011

Liva sover fra kl. 23:30 til kl. 07:45. Vi kan bruge floor() og modulo til at beregne henholdsvis timer og minutter.

Først regner vi ud hvor mange minutter hun sover

$$ { {sleep := 30 + 45 + 7 * 60} } $$

Derefer bruger vi floor() og mod() til at beregne henholdsvis timer og resterende minutter

$$ { { floor({sleep \over 60}) = 8} } $$ $$ {{ mod(sleep ; 60) = 15}}$$

Liva sover i 8 timer og 15 min

Ceiling funktionen

ceiling(x) bruges til at afrunde opad til det nærmeste heltal.

Dette returnerer det største heltal, der er større end eller lig med x.

$$ { \text{ceiling(48,9)} } $$

Dette vil runde ned til 49

Round funktionen

roud(x) bruges til at afrunde til det nærmeste heltal.

Dette returnerer det heltal, hvis der afrundes matematisk korrekt.

$$ { \text{round(628,9)} } $$

Dette vil runde ned til 629
