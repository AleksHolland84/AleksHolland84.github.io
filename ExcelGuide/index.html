TÆL.HVIS Guide

I input feltet skal man være opmærksom på hvilken type data man indsætter. Data-typen kan eks være

  • Tekst (string)
  • Tal (int eller float)
  • dato (date - dato-typen er reelt formateret som tekst)

Symbol-beskrivelse

Du kan også ”pege” på data, ved at henvise til en celle. Dette gøres ved at bruge & symbolet, eks. &F2

Symbol og beskrivelse

Symbol Forklaring
& Sammentrækker data i en celle eller data retuneret af en funktion
" " Beskriver data af typen; string, eller samler udtrykket
"*" Asteriks symbolet representerer et ubestemt antal bogstaver
$ Låser en celle i kolonne eller række
! Reference til et Ark i samme Workbook
[ ] Reference til extern Workbook

Ekempler på at refere til data

Referer data til Ark i samme Workbook

$$ { \text{='Ark3'!E6} } $$

Referer data til Ark i andet Workbook

$$ { \text{='[data.xlsx]Ark1'!A1} } $$

Ekempler på TÆL.HVIS

Tæl hvis år (tal) er større eller lig med 2023:

$$ { \text{=TÆL.HVIS($A$2:$A$17;">=2023")} } $$

Eller med reference til tal i celle F2

$$ { \text{=TÆL.HVIS($A$2:$A$17;">="&F2)} } $$

Tæl hvis celle ikke er tom (blank)

$$ { \text{=TÆL.HVIS($B$2:$B$17;"*")} } $$

Tæl hvis navn indeholder "n" til sidst med reference til celle F4, der indeholder n

$$ { \text{=TÆL.HVIS($B$2:$B$17;"*"&F4)} } $$

Tæl hvis dato er større end dato

$$ { \text{=TÆL.HVIS($D$2:$D$17;">12/12/2022")} } $$

Eller tæl hvis dato er større end dato i celle F6

$$ { \text{=TÆL.HVIS($D$2:$D$17;">"&F6)} } $$

Tæl hvis dato er efter i dag

$$ { \text{=TÆL.HVIS($D$2:$D$17;">"&IDAG())} } $$

$$ { \text{=AFRUND.LOFT(2023;5)} } $$
